🎧 Spotify Clone

A beautifully designed Spotify UI Clone built using Flutter. This project replicates the core interface of Spotify, focusing on modern UI design, smooth navigation, and responsive layout.

🚀 Features
🎵 Modern Spotify-inspired UI
🏠 Home screen with featured playlists
🔍 Search screen UI
🎧 Music player screen design
📱 Responsive layout for mobile & web
🎨 Clean dark theme like Spotify
⚡ Smooth navigation between screens
